Bienvenido al curso de Aplicaciones Web

Objetivo del curso



Desarrollar una aplicación web desde cero utilizando un leguaje del lado del cliente (HTML, CSS) y un lenguaje del lado del servidor (PHP), además de integrar al proyecto una base de Datos que será implementada en MySQL.
